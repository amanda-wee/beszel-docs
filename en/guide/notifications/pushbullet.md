# Pushbullet

## URL Format

<span class="bk">pushbullet://**`api-token`**[/**`device`**/#**`channel`**/**`email`**]</span>

## URL Fields

- **Token** (Required)<br>
  URL part: <code>pushbullet://<strong>token</strong>/targets</code>

- **Targets** (Required)<br>
  URL part: <code>pushbullet://token/<strong>targets</strong></code>
