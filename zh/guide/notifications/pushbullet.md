# Pushbullet

## URL 格式

<span class="bk">pushbullet://**`api-token`**[/**`device`**/#**`channel`**/**`email`**]</span>

## URL 字段

- **Token**（必填）<br>
  URL 部分: <code>pushbullet://<strong>token</strong>/targets</code>

- **Targets**（必填）<br>
  URL 部分: <code>pushbullet://token/<strong>targets</strong></code>
